# 代码说明
* 这是JData比赛Rank13队伍的user模型代码，故成绩只有F11值，F12暂缺
* user_item模型融合代码太杂乱，有空整理上传

## 目录说明

* code: 代码文件
* cache： 缓存数据
* data： 原始数据
* user_model : 最终生成的user

## 代码使用说明
* commen.py: 工具函数
* gen_user_data.py: 提取特征，产生训练集和测试集
* user_model.py: 代码运行入口，训练模型，产生结果
